Silences Installation 침묵들

Description 설명

Description 설명

“침묵들”은 “어제 안에 오늘”로 들어오는 입구이다. 이 장치는 방문객들이 전시회를 보고, 전시회가 구체화하고 있는 기억들에 관여할 때 들어올리는 한국 전쟁을 둘러싼 침묵의 장막을 알린다.

"Silences" is the entryway into Still Present Pasts. This installation conveys the shroud of silences surrounding the Korean War that lifts when visitors view the exhibit and engage with the memories it embodies.
